
# 🎮 Tic-Tac-Toe Web Application

This project is a **Tic-Tac-Toe Web Application** developed using **HTML, CSS, and JavaScript**.
It was created as part of my **Web Development Internship Task (SCT_WD_03)**.

The application allows users to play the classic **Tic-Tac-Toe game either against another player or against the computer**, with real-time game state tracking and win detection.

---

## ✨ Key Features

* 🎮 **Player vs Player Mode**
* 🤖 **Player vs Computer Mode**
* 🧠 **Automatic Win Detection**
* 🔄 **Restart Game Button**
* 📊 **Game Status Display** (shows whose turn it is)
* 🎨 **Modern and Attractive UI Design**
* 🖱️ **Interactive Grid with Hover Effects**
* 📱 **Responsive Layout** for desktop and mobile devices

## 🎮 How to Play

1. Choose the **game mode**:

   * Player vs Player
   * Player vs Computer

2. **Player X starts the game**

3. Click on any **empty box** in the grid to place your mark.

4. The game automatically:

   * Switches turns
   * Checks for **winning combinations**

5. The game ends when:

   * A player wins, or
   * The board is completely filled (**Draw**).

6. Click **Restart Game** to start a new match.
